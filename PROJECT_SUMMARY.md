## 🚀 快速开始

### 1. 环境准备
```bash
# 确保已安装
- Node.js 16+
- MySQL 8.0+
- npm 或 yarn
```

### 2. 安装依赖
```bash
npm install
```

### 3. 配置数据库
```bash
# 创建数据库
mysql -u root -p < database.sql
```

### 4. 配置环境变量
```bash
# 复制配置文件
cp config.env .env

# 编辑数据库配置
# DB_HOST=localhost
# DB_USERNAME=your_username
# DB_PASSWORD=your_password
```

### 5. 启动应用
```bash
# Windows
start.bat

# Linux/Mac
chmod +x start.sh
./start.sh

# 或直接使用 npm
npm run start:dev
```

### 6. 访问应用
打开浏览器访问：`http://localhost:3000`
